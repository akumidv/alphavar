# Risk-adjusted ratios — index

↑ Parent: [risk/](../README.md)

Performance per unit of risk. Convention: `Rp` = portfolio return, `Rf` = risk-free rate.

Children:
- [sortino.md](sortino.md) — downside-deviation ratio.

Quick reference (add a file per ratio as needed):
- **Sharpe** `= (Rp − Rf) / σ_p` — excess return per unit of **total** volatility.
  Source: [Investopedia: Sharpe](https://www.investopedia.com/terms/s/sharperatio.asp).
- **Sortino** — like Sharpe but penalizes only **downside** volatility → [sortino.md](sortino.md).
